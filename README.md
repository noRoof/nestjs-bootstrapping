## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository, with some added features by noRoof. These features are common ones in most server applications

### Features

- Docker environment (dev and prod)
- Logger

### Next steps

Next features to be added:

- Swagger
- Configuration
- TypeORM integration
- Users CRUD

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
